--------------------
Software Stack
--------------------
Our project is a web based auctioning system. 

Our tech stack:
- Play! framework version 1.2.4 for Java
- Sammy.js for client side JavaScript framework
- Twitter Bootstrap for HTML + CSS framework
- Postgres Database


--------------------
Important Files
--------------------
round1/                 -- Root directory of the project
round1/app              -- The bulk of the project, models, views, and controllers
round1/conf             -- Configurations for the http routes
round1/public           -- Contains JavaScript, CSS, and images. Twitter Bootstrap and Sammy.js are here


--------------------
Installation and Usage
--------------------
Play! installation
1. Go to http://www.playframework.org/
2. Download Play Version 1.2.4
3. Unzip
4. Add the unzipped folder to $PATH

To run the project
1. Open up Capstone folder "round1"
2. Enter "> play run" into shell
3. Open up "localhost:9000" inside browser
4. Success! (Fingers crossed)
